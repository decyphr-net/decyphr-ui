# Buttons

Buttons have the following properties:

```javascript
text; // string
color; // string
variant; // string (optional)
modifier; // string (optional)
```

Both the `text` and `color` properties are required, while the `variant` and
`modifier` values are optional.

- `text` is the text that will be displayed on the button
- `color` is the color to be used. Note, this is the color based on the variables configured (`primary`, `secondary`) rather than actual color names
- `variant` provides access to other possible variations of the button. For example, `rounded`
- `modifier` modifies the buttons in some way. For example, adding drop shadow (`shadow`)

All other HTML button properties are also available.

In order to use the Button component, import it using the following:

```javascript
import Button from 'components/elements/Button';
```

## Simple square buttons

Using the standard buttons is straightforward. Simply import using the above import statement and use like the following:

```javascript
<Button text="Submit" color="primary" />
```

## Rounded Buttons

Rounded buttons are a variation of the normal

```javascript
<Button text="Submit" color="primary" variant="rounded" />
```

## Buttons with Drop Shadows

Both rounded and square buttons can have drop shadows

```javascript
<Button text="Submit" color="primary" modifier="shadow" />
<Button text="Submit" color="primary" variant="rounded" modifier="shadow" />
```
